## ℹ️ Overview

This website was made in ASP.net to help table tennis players (like myself) in England work out rankings, as the current <a href="https://www.tabletennisengland.co.uk/content/uploads/2025/07/Ranking-Guide-Aug-2025-v2.pdf">guide</a> is incredibly complex and difficult to understand. Many a time, people have mentioned wondering how a match would affect their ranking, and now, at long last, there is a solution.

## ⬇️ Installation

### Easy Way
1) <a href="https://github.com/ilovepino8x7/ranking-calculator/releases/tag/release">Download the release</a> for your device / OS (comes with .NET pre-installed)
2) Extract the folder - it should make a folder called "Publish"
3) If on Windows, double click the file called "Rankings Calculator Page.exe" and give it permissions if prompted
4) On Mac or Linux, navigate to the folder then run "chmod -R 755 ." and "chmod +x "Rankings Calculator Page"" and then either double click the binary, or use ./"Rankings Calculator Page"

### If a release for your machine is unavailible / release doesn't work
1) Download the repo
2) Install the dotnet 9.0 runtime
3) Navigate to the folder in the repo containing "Ranking Calculator Page.sln" and the folder containing all the data
4) Run the command "dotnet run"
5) Use a browser to navigate to localhost:PORTNUMBER, replacing PORTNUMBER with the output from the terminal.
